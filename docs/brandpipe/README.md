# Brandpipe Docs

This is the supported documentation surface for the repo.

## Keep These
- `docs/brandpipe/run_guide.md`: how to run the generation lane with tracked configs.
- `docs/brandpipe/validation_workflow.md`: how to run the validation lane and interpret outputs.
- `docs/brandpipe/local_remote_ai_workflow.md`: how to choose between fixture, local LM Studio, and OpenRouter-backed runs.

## Supported Paths
- `src/brandpipe/`: the only supported runtime source.
- `resources/brandpipe/`: supported configs, prompts, and fixture data.
- `scripts/brandpipe/`: supported helper utilities.
- `test_outputs/brandpipe/`: generation and validation outputs.

## Historical Material
- `docs/archive/branding/2026/` and `artifacts/branding/legacy/` are historical only.
- They are not part of the supported operator surface and should not be used as current guidance.
